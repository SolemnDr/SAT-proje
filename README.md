# _Sistem Analizi ve Tasarım dönem projesi_
## Oyun Platformu 🎮

Bu proje, kapsamlı bir dijital oyun dağıtım platformunun temel sistem mimarisini ve backend yapısını içermektedir.

## 📌 Modüller ve Özellikler

Sistem yapısı birbirine entegre çalışan 5 ana modül üzerinden kurgulanmıştır:

1. Kullanıcı kaydı Modülü: Kayıt, giriş, güvenlik yönetimi
    1. Şifre kural kontrol var.
    2. Login ve kayıt olma.
    3. Hesap varlık kontrol. Aynı bilgilerle hesap oluşmasın dikkat.
    4. Şifreler güvenli saklanıyor.
2. Mağaza ve Katalog Modülü: Oyunların kategorize edilmesi, arama/filtreleme yöntemleri
    1. Oyun listeleme, arama, filtreleme _olabilir_.
    2. İndirim, çok satanlar gibi filtreleme sistemleri
    3. Tavsiye sistemi _olabilir_.
    4. Sepet sistemi, ödeme sistemi, ödeme bilgileri girme
    5. Sahiplik kontrol sistemi, sahip olunan oyunlar tekrar alınamasın.
    6. Sahip olduğun oyunlara göre öneri sistemi.
3. Kütüphane Modülü: Satın alınan oyunların görüntülenmesi
    1. İndirme kenara ayrıldı. _Olabilir_.
    2. Liste halinde gözükecek.
    3. Yeni listeler oluşturma oyunları kendi isteğine göre listeleme.
    4. Ekleme silme gizleme.
4. Arkadaşlık ve Sosyal Etkileşim Modülü: Arkadaş ekleme ve iletişim kurma
    1. Arkadaş ekleme silme, profil aratma
    2. Hangi arkadaşlar hangi oyunlara sahip göstergesi _olabilir_.
    3. Arkadaş listesi gösterme
    4. Profil fotoğraflar görseller.
5. Oyun Detay ve Bilgilendirme Modülü: Sistem gereksinimleri, oyun içerikleri, yorumlar gibi oyun hakkında bilgilerin verilmesi
    1. Oyunlar için detaylı bilgiler, kategorileri
    2. Kullanıcı yorumları, yıldız verme gibi şeyler
    3. Çıkış tarihi, üretici gibi oyun hakkında bilgiler.
    4. Hangi arakadaşın oyunlara sahip gösterme
    5. Sistem gereksinimleri ilk adımda yok. _ARAŞTIR_.
    6. Görsel Koyma. _ARAŞTIR_.
